# Face-Liveness-Detection

## How to run

git clone https://github.com/EmaanBashir/Face-Liveness-Detection.git

python -m venv venv

cd face-liveness-detection

pip install -r requirements.txt

cd static

mkdir data

cd ..

python manage.py runserver
